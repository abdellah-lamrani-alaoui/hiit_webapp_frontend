# Hiit Webapp Front End

Frontend for the HIIT web app that randomy propose to you a workout to do.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
